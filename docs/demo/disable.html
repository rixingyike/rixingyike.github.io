    干净的页面：功能禁用

    干净的页面：功能禁用

    你可以通过设置页面的 Frontmatter，在页面禁用功能与布局。

    本页面就是一个示例，禁用了如下功能:

    • 导航栏
    • 侧边栏
    • 路径导航
    • 页面信息
    • 贡献者
    • 编辑此页链接
    • 更新时间
    • 上一篇/下一篇 链接
    • 评论
    • 页脚
    • 返回顶部按钮

    禁用代码：

    navbar: false
sidebar: false

breadcrumb: false
pageInfo: false
contributors: false
editLink: false
lastUpdated: false
prev: false
next: false
comment: false
footer: false

backtotop: false
